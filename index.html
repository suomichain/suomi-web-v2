concept

SUOMI means “Finland” in Finnish; Chinese transliteration锁密 denotes “Lock the secret“ in English.
The essence of SUOMI is a random number generator project based on cryptology, expressed by Public Key and Private Key of The Lock.

mission

To create CHAIN of CHAINS, providing a "Blockchain as a service" platform for public, consortium and private chains.

vision

To establish a blockchain-technology based, highly-trusted and high throughout global ecosystem for enterprises and society.

A response time based consensus; Any environments (no BFT)

To reach consensus via multiple rounds votes; For smaller network such as consortium blockchain

Key Technologies

Fast Proof of Work

Four different node types:

General > Candidate > Leader > Restricted

Consesus consists of three phases:

  • 1

    Sign up phase

    General node could register as candidate node

  • 2

    Election phase

    Every candidate requests a random number via VRG

    The candidate who has the smallest number would be elected as leader to publish the new block

  • 3

    Restricted phase

    The leader needs to wait a random time from VRG after publishing the new block

    After the waiting period, the restricted node could register as candidate node

SUOMI Transaction Processing Engine(STPE)

Support stateless UTXO and stateful models.

Support both Ethereum Smart Contracts via EVM and SDKs.

Allow developers to develop their own smart contracts in many popular languages:

  • Interpreted language: Python, JavaScript
  • Compiled language: C + +, Go
  • Virtual Machine: EVM, Java
